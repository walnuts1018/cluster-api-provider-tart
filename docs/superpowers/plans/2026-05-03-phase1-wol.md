# Phase 1 WoL Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** TartMachine が TartHost を予約したあと、対象ホストへ Wake-on-LAN Magic Packet を送信して Phase 1 のインベントリ管理と電源操作を完了する。

**Architecture:** WoL パケット生成と送信は `pkg/wol` に分離し、コントローラーからは小さな interface 経由で呼び出す。TartMachine Reconcile はホスト予約、トークン生成、WoL 送信、status 遷移を一つの状態遷移として扱う。

**Tech Stack:** Go、controller-runtime、Ginkgo/Gomega、標準ライブラリ `net`。
